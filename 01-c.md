# 01-c: 値型（ `struct` ）

今度はクラスではなく値型（ `struct` ）にしてみましょう。

```swift
struct Character {
    let name: String // 名前
    var hp: Int      // HP
    var mp: Int      // MP
    let attack: Int  // 攻撃力
    let defense: Int // 防御力
    ...
}
```

イミュータブルである必要はないので、 01-b のときと違って `hp` と `mp` は `var` で宣言されています。
